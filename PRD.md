# EduHub - 학습 웹앱 포털

## 1. 프로젝트 개요

### 프로젝트명
EduHub (가칭)

### 목적
학생과 교사가 다양한 학습용 웹앱을 쉽고 빠르게 찾고 사용할 수 있도록 하나의 포털에서 제공한다.

단순 링크 모음이 아니라 **검색, 추천, 카테고리, 즐겨찾기, 최근 사용, 태그 기반 탐색** 기능을 제공하는 웹앱 허브를 목표로 한다.

## 20. 관리자 시스템 (Admin)

### 20.1 목적
별도 백엔드 없이 **관리자 코드 입력만으로** 웹앱 카드(링크, 썸네일, 카테고리 등)를 추가/수정/삭제할 수 있도록 한다.

### 20.2 진입 방식
- 사이드바 `⚙ 설정` 하단의 **숨겨진 진입점** 또는 URL `/admin` 직접 접근
- 진입 시 **관리자 코드 입력 모달** 노출
- 코드 일치 시 관리자 모드 활성화 (세션 유지)

### 20.3 관리자 코드
- 기본값: `EDUHUB-ADMIN-2026` (배포 시 환경변수 또는 `config.js`에서 변경 가능)
- 저장 위치: `localStorage.eduhub_admin_token`
- 로그아웃 버튼으로 토큰 제거

```js
// config.js 예시
export const ADMIN_CODE = "EDUHUB-ADMIN-2026";
```

### 20.4 관리자 화면 구성

```
┌──────────────────────────────────────────────┐
│ Admin Panel                          [로그아웃]│
├──────────────────────────────────────────────┤
│ [+ 새 웹앱 추가]   [카테고리 관리] [Export]   │
├──────────────────────────────────────────────┤
│ # | 썸네일 | 이름     | 카테고리 | 링크 | 수정/삭제 │
│ 1 | [img] | ChatGPT | AI      | URL  | ✏ 🗑     │
│ 2 | [img] | Kahoot  | 퀴즈    | URL  | ✏ 🗑     │
└──────────────────────────────────────────────┘
```

### 20.5 웹앱 등록 폼

| 필드 | 타입 | 필수 | 설명 |
|---|---|---|---|
| 앱 이름 | text | ✅ | 카드에 표시되는 이름 |
| 한줄 설명 | text | ✅ | 카드 한 줄 요약 (60자 이내) |
| 상세 설명 | textarea | - | 상세페이지용 |
| 링크(URL) | url | ✅ | 바로가기 대상 |
| 썸네일/아이콘 | file 또는 URL | ✅ | 이미지 업로드 → Base64 인코딩 후 저장 |
| 카테고리 | select | ✅ | AI / 퀴즈 / 평가 / 영상 / 협업 등 |
| 태그 | multi-select | - | `#무료`, `#초등`, `#수학` 등 |
| 교육대상 | checkbox | - | 초등/중등/고등/교사/특수 |
| 가격 | radio | - | 무료 / 유료 / 부분유료 |
| 한국어 지원 | toggle | - | Y / N |
| 추천 여부 | toggle | - | Hero/추천 섹션 노출 |

### 20.6 썸네일 업로드 처리
- 파일 input → `FileReader.readAsDataURL()` → Base64 변환
- 변환된 문자열을 JSON 데이터에 함께 저장
- **이미지 권장 사양**: 정사각형 256x256, 500KB 이하 (초과 시 캔버스로 리사이즈)
- URL 직접 입력도 허용 (외부 이미지 호스팅 사용 시)

```js
function handleThumbnail(file) {
  const reader = new FileReader();
  reader.onload = e => saveApp({ thumbnail: e.target.result });
  reader.readAsDataURL(file);
}
```

### 20.8 카테고리 관리
- 관리자 패널에서 카테고리 추가/이름변경/삭제 가능
- 기본 카테고리는 잠금(삭제 불가)
- 사용자 정의 카테고리는 `localStorage.eduhub_categories` 에 저장

### 20.9 Export / Import
- **Export**: 전체 데이터를 `eduhub-backup-YYYYMMDD.json` 파일로 다운로드
- **Import**: 백업 JSON 업로드 시 기존 데이터 덮어쓰기 또는 병합 선택
- 배포 시 기본 데이터(`seed.json`)를 함께 동봉

### 20.10 권한 분리

| 작업 | 일반 사용자 | 관리자 |
|---|---|---|
| 카드 보기 | ✅ | ✅ |
| 즐겨찾기 | ✅ | ✅ |
| 카드 추가 | ❌ | ✅ |
| 카드 수정 | ❌ | ✅ |
| 카드 삭제 | ❌ | ✅ |
| 카테고리 관리 | ❌ | ✅ |
| Export/Import | ❌ | ✅ |

### 20.11 유효성 검사
- URL 형식 검사 (http/https)
- 필수 항목 미입력 시 저장 불가
- 동일 이름 중복 시 경고 모달
- 썸네일 용량 초과 시 자동 리사이즈

### 20.12 보안 주의
- 클라이언트 전용 코드 인증은 **간이 보호** 수준임을 명시
- 실서비스 단계(v2)에서는 Firebase Auth + Firestore 등 백엔드 인증으로 전환 권장
